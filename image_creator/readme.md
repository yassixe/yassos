## Image Creator
The goal of this program is to create a hard disk image and place our uefi os loader on it and to do so i should respect the GPT partition table.
    
    sources:
        1- uefi spec.
        2- gnu-efi.
        3- a tutorial on youtube (Queso Fuego).
